# WezTerm config files

## Install

### Powershel

```powershell
git clone https://github.com/JonasGao/weztermrc.git "$HOME\.config\wezterm"
```

Or ssh

```powershell
git clone git@github.com:JonasGao/weztermrc.git "$HOME\.config\wezterm"
```

### Linux / MacOS

```shell
git clone https://github.com/JonasGao/weztermrc.git $HOME/.config/wezterm
```

Or ssh

```shell
git clone git@github.com:JonasGao/weztermrc.git $HOME/.config/wezterm
```
